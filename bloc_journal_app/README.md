# Little Moments Journal - BLoC + Dio

A Flutter journal app using **BLoC** state management and **Dio** HTTP client.

##  Features

- **READ** - Fetch quotes from DummyJSON API using Dio
- **CREATE** - Add custom quotes to journal
- **UPDATE** - Edit personal notes and mark favorites
- **DELETE** - Remove quotes with swipe gesture
- Loading & error states
- BLoC event-driven architecture

## 🛠️ Tech Stack

- Flutter
- flutter_bloc (State Management)
- dio (HTTP Client)
- equatable (Value Equality)



## Getting Started
